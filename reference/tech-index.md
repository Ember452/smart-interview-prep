## 查找规则

1. 解析简历后，提取技术栈关键词和项目类型
2. 按关键词匹配索引表，确定需要加载的域文件
3. 每个知识库文件按子题分层（如"数据库 → 索引 → B+Tree 索引结构"），面试官自由选题
4. 题目难度根据候选人实际回答质量和面试官风格动态调整
5. 知识库内容为答案要点参考，面试官据此评估回答质量并结合回答内容即兴追问
