# 语种适配规则

本文件包含语种适配规则和简历信息缺失兜底策略；执行时以 `SKILL.md` 为准。

| language 值 | 行为 |
|------------|------|
| auto | 自动检测用户输入语言，全程跟随该语种提问和输出报告 |
| zh-CN | 全程中文提问、中文报告 |
| en-US | 全程英文提问、英文报告（报告维度不变但内容英文化） |

**语种切换仅允许在面试开始前执行，面试进行中禁止切换语言。**

# 简历信息缺失兜底

| 缺失情况 | 兜底策略 |
|---------|---------|
| 无技术栈 | 不出专业技术题，仅输出通用行为面试题（团队协作、解决问题能力、学习能力等），并提示："未识别到具体技术栈，已为你生成通用面试题。如需技术面试，请补充相关技能信息。" |
| 无项目经历 | 项目深挖占比降为 0，增加技术基础题和场景假设题（"如果让你设计一个XXX系统，你会怎么做？"），并提示："未发现具体项目经历，将侧重技术基础和场景设计题。" |
| 无工作经历 | 行为面试占比降为 0，侧重技术基础和项目经验（如有），并提示："未识别到工作经历，已调整面试侧重技术能力。" |
| 仅教育背景 | 以基础题和课程/实习项目为主，同时仍可涉及简历中提到的任何技术方向 |
